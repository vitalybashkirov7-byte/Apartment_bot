Использовать Karing как системный прокси (проще)
Если Karing уже работает и через него открываются сайты, можно просто сказать боту использовать системные настройки прокси.
В коде бота (в parser.py) измени:
python
import requests
# Вместо явного указания прокси, используйте системные настройки
response = requests.get(url, timeout=30)  # без proxies={}
Но это не всегда работает, потому что Karing может использовать SOCKS5, а requests не всегда корректно работает с ним через системные настройки.
